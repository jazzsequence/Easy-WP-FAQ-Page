## Description ##

Easily add Frequently Asked Questions to any WordPress Page!

## Installation ##

### Manual Installation ###

1. Upload the entire `/wds-faq-page` directory to the `/wp-content/plugins/` directory.
2. Activate WDS FAQ Page! through the 'Plugins' menu in WordPress.
